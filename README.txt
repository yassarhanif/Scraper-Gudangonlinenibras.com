1. Isi file excel "link yang mau discrape.csv" dengan link dari GON. Taruh link secara menurun.

2. Buka file "MULAI SCRAPING.bat" dan proses akan berjalan. Tunggu sampai muncul tulisan:

"Tahap 2: Sortir gambar yang ingin diupload ke Tiktok. Masuk ke folder 'Source Code/HASIL SCRAPE/' lalu pilih 8 gambar yang ingin diupload ke tiktok. Maksimal 8 gambar dan tidak boleh lebih"

3. Jika sudah selesai menyortir gambar, kembali ke window "MULAI SCRAPING.bat" dan pencet tombol ENTER untuk melajutkan proses scraping. Tunggu lagi prosesnya.

4. Jika sudah selesai, akan muncul "PROSES SCRAPING SUDAH SELESAI".

5. Pencet tombol apapun untuk menutup window "MULAI SCRAPING.bat"

6. Cari file "hasil.xlsx" di folder Scraper GON
